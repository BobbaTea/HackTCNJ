# HackTCNJ 2019

## Running the Server
- Run these commands in this order: 
- `python3 -m venv venv`
- `source venv/bin/activate`
- `pip install -r requirements.txt`
- `npm install`
- `./run.sh`
- From now on, just run `./run.sh` and you'll be good
